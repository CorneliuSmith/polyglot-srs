  <behavior>
    ## Normalization (NLP-03)
    - normalize("Собака") -> "собака" (lowercase only)
    - normalize("  привет  ") -> "привет" (strip whitespace)

    ## Lemmatization (NLP-03)
    - lemmatize("собаку") -> "собака" (accusative -> nominative)
    - lemmatize("пишет") -> "писать" (conjugated -> infinitive)
    - lemmatize("красивого") -> "красивый" (adjective inflection -> base form)

    ## Morphological Family (NLP-03)
    - get_morphological_family("собака") -> set containing at least {"собака", "собаки", "собаке", "собаку", "собакой"}
    - All forms returned are lowercase strings

    ## Transliteration (NLP-04)
    - check_answer("sobaka", "собака") -> (CORRECT_SLOPPY, message about using Cyrillic)
    - check_answer("da", "да") -> (CORRECT_SLOPPY, message about Cyrillic)
    - check_answer("sobaka", "кошка") -> does NOT return CORRECT_SLOPPY (transliteration doesn't match)
    - Only triggers when input is ASCII (not when input is already Cyrillic)

    ## Aspect Partner Detection (NLP-05)
    - get_aspect_partner("писать", {"morphology": {"aspect_partner": "написать"}}) -> "написать"
    - get_aspect_partner("написать", {"morphology": {"aspect_partner": "писать"}}) -> "писать"
    - get_aspect_partner("собака", None) -> None (not a verb / no partner data)
    - check_answer("писать", "написать", {"morphology": {"aspect_partner": "писать"}}) -> (WRONG_FORM, message about aspect)

    ## Full Pipeline Integration
    - check_answer("собака", "собака") -> (CORRECT, None) — exact match
    - check_answer("Собака", "собака") -> (CORRECT, None) — normalized match
    - check_answer("собаку", "собака") -> (CORRECT_SLOPPY, message) — same lemma, different case
    - check_answer("кошка", "собака") -> (WRONG, None) — completely wrong
  </behavior>
  <implementation>
    Create `backend/services/nlp/russian.py` with class `RussianNLP(BaseNLP)`:

    1. Module-level `morph = pymorphy3.MorphAnalyzer()` — loaded once (singleton pattern per research).

    2. `normalize(text)`: strip + lowercase. No diacritic stripping needed for Russian.

    3. `lemmatize(word)`: `morph.parse(word.lower())[0].normal_form`. Use first parse result (highest score).

    4. `get_morphological_family(word)`: `morph.parse(word.lower())[0].lexeme` -> extract all `.word` values as a set.

    5. `get_aspect_partner(verb, card_context)`: Look up `card_context["morphology"]["aspect_partner"]` if available. Return None if card_context is None or key doesn't exist. Do NOT try to compute aspect partners from pymorphy3 — it doesn't have them (research pitfall #2).

    6. Override `check_answer()`: Before calling `super().check_answer()`, check if input is ASCII. If so, transliterate via `cyrtranslit.to_cyrillic(input, "ru")` and compare normalized result to normalized correct answer. If match, return (CORRECT_SLOPPY, "Correct! Use Cyrillic next time."). Otherwise fall through to super().

    Install pymorphy3 and cyrtranslit: `pip install pymorphy3 cyrtranslit`
  </implementation>
