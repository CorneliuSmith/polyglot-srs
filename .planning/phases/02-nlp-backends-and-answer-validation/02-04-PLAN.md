  <behavior>
    ## Normalization (NLP-09)
    - normalize("The Dog") -> "dog" (lowercase + strip leading article)
    - normalize("A cat") -> "cat" (strip 'a')
    - normalize("An apple") -> "apple" (strip 'an')
    - normalize("  hello  ") -> "hello" (strip whitespace)
    - normalize("dog") -> "dog" (no article, just lowercase)
    - normalize("the") -> "the" (edge case: don't strip if the entire word IS the article)

    ## Lemmatization (NLP-09)
    - lemmatize("went") -> "go" (irregular verb)
    - lemmatize("mice") -> "mouse" (irregular noun)
    - lemmatize("cats") -> "cat" (regular noun)
    - lemmatize("running") -> "run" (gerund)
    - lemmatize("better") -> "well" or "good" (comparative — accept spaCy's output)

    ## Morphological Family (NLP-09)
    - get_morphological_family("go") -> set containing at least {"go", "goes", "going", "went", "gone"}
    - get_morphological_family("mouse") -> set containing at least {"mouse", "mice"}
    - get_morphological_family("cat") -> set containing at least {"cat", "cats"}
    - Uses lemminflect.getAllInflections, NOT naive string concatenation (research anti-pattern)

    ## Aspect Partner (NLP-09)
    - get_aspect_partner("go", None) -> None (English has no aspect partner system)

    ## Full Pipeline
    - check_answer("cat", "cat") -> (CORRECT, None)
    - check_answer("Cat", "cat") -> (CORRECT, None) — normalized
    - check_answer("the cat", "cat") -> (CORRECT, None) — article stripped
    - check_answer("cats", "cat") -> (CORRECT_SLOPPY, message) — same lemma
    - check_answer("went", "go") -> (CORRECT_SLOPPY, message) — same lemma, different inflection
    - check_answer("dog", "cat") -> (WRONG, None)
    - check_answer("color", "colour", {"answer_alternatives": ["color"]}) -> (CORRECT, None) — alternatives
  </behavior>
  <implementation>
    Create `backend/services/nlp/english.py` with class `EnglishNLP(BaseNLP)`:

    1. Module-level initialization:
       ```python
       import spacy
       import lemminflect  # Must import to register spaCy extensions
       _nlp = spacy.load("en_core_web_sm")
       ```
       Wrap in try/except — if spaCy model not downloaded, log warning and set _nlp to None.

    2. `normalize(text)`:
       - Strip whitespace, lowercase
       - Strip leading articles: if text starts with "the ", "a ", or "an ", remove the article prefix
       - Edge case: if text IS just "the", "a", or "an" (no space after), do NOT strip (the word itself is the answer)

    3. `lemmatize(word)`:
       - If _nlp is None, return word.lower()
       - Process with spaCy: `doc = _nlp(word.lower()); return doc[0].lemma_`
       - spaCy handles irregular verbs, nouns via its lemmatizer

    4. `get_morphological_family(word)`:
       - Lemmatize first to get base form
       - Use `lemminflect.getAllInflections(lemma)` to get all inflected forms
       - Flatten all values (tuples) from the dict into a single set
       - Add the original word and lemma to the set
       - If lemminflect not available, fallback to {word, lemma} minimal set

    5. `get_aspect_partner(verb, card_context)`: Return None. English has no aspect partner system.

    No need to override check_answer — the base pipeline handles everything for English.

    Install: `pip install spacy lemminflect && python -m spacy download en_core_web_sm`
  </implementation>
</feature>
