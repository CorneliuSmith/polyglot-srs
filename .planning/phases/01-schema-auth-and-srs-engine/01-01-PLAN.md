Create three migration files:

**Migration 1: 20260312000000_initial_schema.sql** -- All table DDL:

1. **languages** table:
   - id UUID PK DEFAULT gen_random_uuid()
   - code TEXT NOT NULL UNIQUE (e.g., 'ru', 'ar', 'en')
   - name TEXT NOT NULL (e.g., 'Russian')
   - rtl BOOLEAN DEFAULT false
   - created_at TIMESTAMPTZ DEFAULT now()

2. **grammar_points** table:
   - id UUID PK DEFAULT gen_random_uuid()
   - language_id UUID NOT NULL REFERENCES languages(id)
   - title TEXT NOT NULL
   - explanation TEXT
   - level TEXT CHECK (level IN ('A1','A2','B1','B2','C1','C2'))
   - display_order INT DEFAULT 0
   - prerequisites UUID[] DEFAULT '{}' (array of grammar_point IDs for prerequisite chains)
   - created_at TIMESTAMPTZ DEFAULT now()
   - INDEX on (language_id, level, display_order)

3. **vocabulary** table:
   - id UUID PK DEFAULT gen_random_uuid()
   - language_id UUID NOT NULL REFERENCES languages(id)
   - word TEXT NOT NULL
   - reading TEXT (pronunciation guide -- furigana, transliteration, tashkeel form)
   - part_of_speech TEXT
   - level TEXT CHECK (level IN ('A1','A2','B1','B2','C1','C2'))
   - frequency_rank INT
   - morphology JSONB DEFAULT '{}' (stores per-language grammar features: gender, aspect, root, form, declension, etc.)
   - alternatives TEXT[] DEFAULT '{}' (regional spellings, archaic forms, aspect partners for NLP-10)
   - created_at TIMESTAMPTZ DEFAULT now()
   - INDEX on (language_id, level)
   - INDEX on (language_id, frequency_rank)

4. **translations** table:
   - id UUID PK DEFAULT gen_random_uuid()
   - vocabulary_id UUID NOT NULL REFERENCES vocabulary(id) ON DELETE CASCADE
   - locale TEXT NOT NULL (en, ru, ar, es, pt)
   - definition TEXT NOT NULL
   - created_at TIMESTAMPTZ DEFAULT now()
   - UNIQUE(vocabulary_id, locale)
   - INDEX on vocabulary_id

5. **drill_sentences** table:
   - id UUID PK DEFAULT gen_random_uuid()
   - grammar_point_id UUID NOT NULL REFERENCES grammar_points(id) ON DELETE CASCADE
   - vocabulary_id UUID REFERENCES vocabulary(id) ON DELETE SET NULL (optional link)
   - sentence TEXT NOT NULL (contains {{answer}} marker)
   - translation TEXT (English translation of the sentence)
   - hint TEXT (optional hint shown to user)
   - display_order INT DEFAULT 0
   - created_at TIMESTAMPTZ DEFAULT now()
   - INDEX on grammar_point_id

6. **content_lists** table:
   - id UUID PK DEFAULT gen_random_uuid()
   - language_id UUID NOT NULL REFERENCES languages(id)
   - list_type TEXT NOT NULL CHECK (list_type IN ('grammar', 'vocabulary'))
   - level TEXT CHECK (level IN ('A1','A2','B1','B2','C1','C2'))
   - title TEXT NOT NULL
   - description TEXT
   - display_order INT DEFAULT 0
   - created_at TIMESTAMPTZ DEFAULT now()
   - UNIQUE(language_id, list_type, level)

7. **user_profiles** table:
   - id UUID PK REFERENCES auth.users(id) ON DELETE CASCADE
   - batch_size INT DEFAULT 5 CHECK (batch_size BETWEEN 1 AND 50)
   - ui_language TEXT DEFAULT 'en'
   - active_language_id UUID REFERENCES languages(id)
   - created_at TIMESTAMPTZ DEFAULT now()
   - updated_at TIMESTAMPTZ DEFAULT now()

8. **user_cards** table:
   - id UUID PK DEFAULT gen_random_uuid()
   - user_id UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE
   - language_id UUID NOT NULL REFERENCES languages(id)
   - card_type TEXT NOT NULL CHECK (card_type IN ('grammar', 'vocabulary'))
   - card_id UUID NOT NULL (polymorphic FK to grammar_points or vocabulary)
   - ease_factor FLOAT DEFAULT 2.5
   - interval INT DEFAULT 1
   - repetitions INT DEFAULT 0
   - streak INT DEFAULT 0
   - lapses INT DEFAULT 0
   - next_review TIMESTAMPTZ DEFAULT now()
   - last_review TIMESTAMPTZ
   - is_suspended BOOLEAN DEFAULT false
   - created_at TIMESTAMPTZ DEFAULT now()
   - UNIQUE(user_id, card_type, card_id)
   - INDEX on (user_id, language_id, next_review)
   - INDEX on (user_id, language_id, is_suspended)

9. **review_log** table:
   - id UUID PK DEFAULT gen_random_uuid()
   - user_id UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE
   - card_id UUID NOT NULL REFERENCES user_cards(id) ON DELETE CASCADE
   - quality INT NOT NULL CHECK (quality BETWEEN 0 AND 5)
   - ease_factor_before FLOAT NOT NULL
   - ease_factor_after FLOAT NOT NULL
   - interval_before INT NOT NULL
   - interval_after INT NOT NULL
   - time_taken_ms INT (milliseconds to answer)
   - answer_result TEXT (CORRECT, CORRECT_SLOPPY, WRONG_FORM, WRONG)
   - created_at TIMESTAMPTZ DEFAULT now()
   - INDEX on (user_id, created_at)

10. **user_content_subscriptions** table:
    - id UUID PK DEFAULT gen_random_uuid()
    - user_id UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE
    - content_list_id UUID NOT NULL REFERENCES content_lists(id) ON DELETE CASCADE
    - subscribed_at TIMESTAMPTZ DEFAULT now()
    - UNIQUE(user_id, content_list_id)

**Migration 2: 20260312000001_rls_policies.sql** -- RLS policies:

For each user table, enable RLS and create per-operation policies:

- **user_profiles**: SELECT (USING auth.uid()=id), INSERT (WITH CHECK auth.uid()=id), UPDATE (USING + WITH CHECK auth.uid()=id). No DELETE policy (cascade handles it).
- **user_cards**: SELECT, INSERT, UPDATE, DELETE -- all scoped to auth.uid()=user_id.
- **review_log**: SELECT (USING auth.uid()=user_id), INSERT (WITH CHECK auth.uid()=user_id). No UPDATE or DELETE (append-only).
- **user_content_subscriptions**: SELECT, INSERT, DELETE -- scoped to auth.uid()=user_id. No UPDATE (delete and re-insert).

Public tables (languages, grammar_points, vocabulary, translations, drill_sentences, content_lists) do NOT get RLS enabled -- they are public content readable by all authenticated users. This avoids Pitfall 4 from research.

Use ON CONFLICT (code) DO NOTHING for idempotency.
