1. Create `frontend/src/features/review/useReviewSession.ts`:
   - Takes `cards: DueCard[]` as input
   - State: `queue` (the cards array), `currentIndex` (number), `results` (array of session results), `phase` ('answering' | 'feedback' | 'rating' | 'summary')
   - `currentCard`: queue[currentIndex] or null
   - `cardStartTime`: useRef<number> tracking when current card was shown
   - `setFeedback(result: ValidateAnswerResponse)`: stores NLP feedback, transitions phase to 'feedback'
   - `recordResult(answerResult: string, timeTakenMs: number)`: appends to results array, transitions to 'rating'... actually, the flow should be:
     Phase flow: answering -> (user submits answer) -> feedback (NLP result shown) -> (user clicks rating) -> next card or summary
     So: answering -> user types + presses Enter/Submit -> call validateAnswer API -> feedback phase -> user sees result + clicks Again/Hard/Good/Easy -> call submitReview API -> advance to next card (answering) or summary
   - `submitAnswer(userInput: string)`: called when user presses Enter/submit. Returns void -- the page component handles the API call and passes the result back via `setValidationResult`
   - `setValidationResult(result: ValidateAnswerResponse)`: stores the NLP response, transitions to 'feedback'
   - `rate(answerResult: string)`: records the rating result in results array, advances to next card or summary
   - `advance()`: increments currentIndex, resets phase to 'answering', resets cardStartTime
   - `elapsedMs()`: returns Date.now() - cardStartTime.current
   - Session stats computed from results: `accuracy` (count of correct+correct_sloppy / total), `totalTimeMs`, `cardsReviewed`
   - isComplete: currentIndex >= queue.length
