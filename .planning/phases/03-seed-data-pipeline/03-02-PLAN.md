# Plan 03-02: Arabic Seeder

**Phase**: 03 — Seed Data Pipeline
**Requirements**: SEED-02
**Wave**: 2 (parallel with 03-03, after 03-01)
**Depends on**: 03-01 (BaseSeeder infrastructure)

## Goal

Create the Arabic seed script that loads curated Arabic vocabulary with root extraction, morphology JSONB, and frequency ranking into the vocabulary + translations tables.

## Success Criteria

1. Arabic vocabulary is loaded with word, reading (tashkeel form), POS, frequency_rank, root in morphology JSONB
2. Morphology JSONB includes: root, pattern, form (verb form I-X), gender, number where available
3. Translations are loaded into translations table with locale='en'
4. Script handles missing camel_data gracefully (falls back to manually provided morphology in seed file)
5. Seed data covers A1-B1 (~500+ high-frequency words)

## Tasks

### Task 1: Curated Arabic seed data file

Create `data/ar_seed.json` — a curated JSON file with ~500 high-frequency Arabic words:

```json
[
  {
    "word": "كتب",
    "reading": "كَتَبَ",
    "pos": "verb",
    "rank": 15,
    "root": "كتب",
    "form": "I",
    "translations": {"en": "to write"}
  },
  {
    "word": "كتاب",
    "reading": "كِتَاب",
    "pos": "noun",
    "rank": 42,
    "root": "كتب",
    "gender": "m",
    "translations": {"en": "book"}
  }
]
```

**Why curated instead of Arabic Wordnet?** Arabic Wordnet XML is complex, URLs are unstable, and the data quality for frequency ranking is poor. A curated seed of 500 high-frequency words with pre-verified roots and forms produces better learning content. This can be expanded later.

Include words from these categories:
- Top 100 verbs (Forms I-X represented)
- Top 100 nouns (masculine/feminine, sound/broken plurals)
- Top 50 adjectives
- Top 50 particles/prepositions
- Common greetings and phrases

### Task 2: Arabic seeder implementation

Create `backend/services/seeder/seed_arabic.py`:

```python
class ArabicSeeder(BaseSeeder):
    language_code = "ar"

    async def download(self) -> None:
        """Arabic uses a bundled seed file — no download needed."""
        seed_path = DATA_DIR / "ar_seed.json"
        if not seed_path.exists():
            raise FileNotFoundError(
                f"Arabic seed file not found at {seed_path}. "
                "Copy data/ar_seed.json to the data directory."
            )

    async def transform(self) -> list[dict]:
        """Parse Arabic seed JSON into vocabulary records."""
        import json
        seed_path = DATA_DIR / "ar_seed.json"
        raw = json.loads(seed_path.read_text(encoding="utf-8"))

        # Optional: enrich with camel-tools
        analyzer = None
        try:
            from camel_tools.morphology.analyzer import Analyzer
            from camel_tools.morphology.database import MorphologyDB
            db = MorphologyDB.builtin_db()
            analyzer = Analyzer(db)
        except (ImportError, Exception):
            self.logger.warning("camel-tools not available — using seed file morphology only")

        records = []
        for item in raw:
            morphology = {
                "root": item.get("root"),
                "form": item.get("form"),
                "gender": item.get("gender"),
                "pattern": item.get("pattern"),
            }

            # Enrich with camel-tools if available
            if analyzer:
                try:
                    analyses = analyzer.analyze(item["word"])
                    if analyses:
                        best = analyses[0]
                        morphology.setdefault("root", best.get("root", ""))
                        morphology.setdefault("pattern", best.get("pattern", ""))
                        morphology.setdefault("form", best.get("vform", ""))
                except Exception:
                    pass

            # Clean None values
            morphology = {k: v for k, v in morphology.items() if v}

            records.append({
                "word": item["word"],
                "reading": item.get("reading"),
                "pos": item.get("pos"),
                "level": self.rank_to_level(item.get("rank")),
                "frequency_rank": item.get("rank"),
                "morphology": json.dumps(morphology, ensure_ascii=False),
                "translations": item.get("translations", {}),
            })

        self.logger.info(f"Transformed {len(records)} Arabic words")
        return records
```
