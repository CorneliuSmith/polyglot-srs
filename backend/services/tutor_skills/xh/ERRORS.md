# Common learner errors — Xhosa

- **Concord chain breaks.** The noun class must echo through the sentence:
  ABAntwana ABAkhulu BAyadlala (class 2). Rebuild the full chain when any
  link breaks; name the class number.
- **Class pair confusion.** um-/aba- (people) vs um-/imi- (trees, some
  objects) share singular shape; the plural and concords differ. Teach the
  pair, not the singular alone.
- **Click substitution.** c/q/x replaced with k or t. Coach by place: c =
  tongue-tip behind teeth ('tsk'), q = roof pop, x = side of tongue. Praise
  approximations; precision follows exposure.
- **Copulative formation.** 'He is a teacher' = NGutitshala (ng- copulative),
  not a be-verb. Drill the copulative prefixes by class.
- **Locative formation.** e-...-ini transforms (indlu → endlwini); learners
  bolt on prepositions instead. Coach the transform as a pattern family.
- **Object concord omission.** Ndiyambona (I see him) — the -m- slot; force
  retrieval with person-object drills.
- **hlonipha awareness.** Respect vocabulary matters in some contexts —
  flag it as register when relevant, don't drill it early.
