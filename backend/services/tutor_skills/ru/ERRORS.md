# Common learner errors — Russian

For each: name the pattern, show the contrast, have them retry a similar item.

- **Aspect swap in the past.** "Я читал книгу вчера" when a finished, one-time
  event needs прочитал. Contrast: читал (process/repeated) vs прочитал
  (completed result). Ask: did the action reach its result?
- **Case after в/на.** Accusative for motion (в школу), prepositional for
  location (в школе). Drill the same noun both directions.
- **Genitive after negation and numbers.** нет времени, два часа/пять часов —
  numbers 2–4 take genitive singular, 5+ genitive plural.
- **Motion verb choice.** идти (one direction, now) vs ходить (round trips,
  habitual). "Я иду в кино каждую неделю" → хожу.
- **Stress misplacement.** Mobile stress changes forms (руки́ gen sg vs ру́ки
  nom pl). Always re-mark stress when correcting.
- **ты/вы mixing mid-conversation.** Point it out as a register slip, not a
  grammar error.
- **Transliteration artifacts.** QWERTY input can produce ш/щ or ь/ъ slips —
  if the Cyrillic looks one key off, coach the typing, not the grammar.
