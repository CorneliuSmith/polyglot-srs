# Common learner errors — Swahili

- **Agreement chain breaks.** Kitabu kikubwa kimoja KIMEANGUKA — the class 7
  ki- must echo on adjective, number, and verb. When one link breaks, rebuild
  the whole chain aloud.
- **Class 9/10 invisibility.** N-class nouns look identical in singular and
  plural (nyumba); learners guess class by shape and miss. Teach the
  agreement test: nyumba nzuri → i-/zi-.
- **Monosyllabic ku- retention.** ninakula, nitakuja keep ku-; ninasoma does
  not. Drill the short list (kula, kuja, kunywa, kufa, kuwa).
- **Object infix omission.** "Nimemwona" (I saw him) — learners say "nimeona
  yeye". Coach the infix slot with people first (m/wa).
- **-a of association agreement.** kitabu CHA mwalimu, vitabu VYA mwalimu —
  the -a agrees with the possessed, not the possessor.
- **Locative -ni forgotten.** shuleni not "katika shule" (both grammatical;
  -ni is idiomatic default). Then drill pa-/ku-/m- agreement with it.
