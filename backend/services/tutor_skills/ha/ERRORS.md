# Common learner errors — Hausa

- **Aspect pronoun-set swaps.** THE core error: na tafi (completed) vs ina
  tafiya (ongoing) vs nakan tafi (habitual) vs zan tafi (future). Coach by
  the sentence's time cue: jiya → completive; yanzu → continuous; kowace
  rana → habitual; gobe → future. Retry with a new time adverb each miss.
- **Gender agreement on possessives/demonstratives.** -n (m) vs -r (f):
  gidansa vs motarsa; wannan agrees. Drill with the learner's own noun bank.
- **Plural overgeneralization.** Applying one broken-plural pattern
  everywhere (gida → gidaje but yaro → yara). Teach patterns as families;
  accept pattern-guessing as a good instinct, then correct.
- **Hooked consonant substitution.** b/ɓ, d/ɗ, k/ƙ are different words in
  writing (daya 'the one' vs ɗaya 'one'). Coach as spelling, model the
  glottalized sound by description.
- **Grade/valence confusion in verbs.** sayar 'sell' vs saya 'buy' — verb
  endings shift meaning; drill in pairs.
- **Skipping greetings.** Pragmatics, not grammar: entering a request without
  the greeting exchange reads as rude. Model the exchange.
