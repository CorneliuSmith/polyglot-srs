# Common learner errors — Greek

- **Aspect stem choice.** The core struggle: θα γράψω (once) vs θα γράφω
  (ongoing/habitual), να φάω vs να τρώω. When wrong, ask: one bounded event,
  or a repeated/ongoing one? Drill the same verb both ways with time
  adverbs.
- **Accent omission.** Monotonic Greek = exactly one accent per polysyllabic
  word, and it's phonemic (πότε/ποτέ, νόμος/νομός). Correct every miss
  gently; accent placement IS spelling.
- **Case after prepositions.** σε/από/με/για take accusative; the genitive
  creeps in from possession drills. Contrast στη Μαρία vs της Μαρίας.
- **Final -ν rules.** την/τη, τον/το, δεν/δε before vowels and stops vs
  continuants — a polish item; coach in writing once basics are solid.
- **Gender-article mismatches.** ο/η/το must match through adjectives:
  η ωραία πόλη. Rebuild the chain on a miss.
- **ς forgotten.** Word-final sigma is ς, not σ — pure orthography, quick
  fix.
- **Transliteration slips.** QWERTY-to-Greek input can yield η/ι/υ or ο/ω
  swaps that LOOK like spelling errors — all sound identical; coach which
  homophone belongs in this word.
