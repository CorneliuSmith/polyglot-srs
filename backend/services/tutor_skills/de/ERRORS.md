# Common learner errors — German

- **Wechselpräposition case.** in/an/auf + accusative (motion) vs dative
  (location): in DIE Schule (going) vs in DER Schule (being). The single
  highest-yield drill; use the same sentence both ways.
- **Adjective endings.** Teach as article+adjective system: der große Mann /
  ein großER Mann. When wrong, identify which slot (article type, case,
  gender), then retry.
- **Verb-final after subordinators.** weil ich müde BIN — learners keep V2
  after weil (common in speech, wrong in standard). Accept it exists, coach
  the standard.
- **Separable prefix placement.** Ich stehe um 6 auf — prefix goes to the
  clause end; in subordinate clauses it reattaches (dass ich aufstehe).
- **Gender guessing.** Always learn Nomen MIT Artikel. Correct gently;
  suggest der/die/das memory hooks from the noun's ending where they exist
  (-ung f, -chen n).
- **Plural formation.** Multiple patterns (-e, -er, ¨-e, -n, -s); drill each
  noun's plural with the noun itself.
- **du/Sie drift.** Register slip, not grammar — flag it and move on.
