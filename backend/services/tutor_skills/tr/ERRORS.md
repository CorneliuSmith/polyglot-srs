# Common learner errors — Turkish

- **Vowel harmony violations.** evlar → evler. Don't just correct: walk the
  rule (front vowel e → -ler). Harmony errors early = system not yet
  internalized; drill harmony pairs until automatic.
- **Consonant mutation misses.** kitap + ı → kitabı (p→b), gitmek → gidiyor.
  Teach the voicing rule with the learner's own vocabulary.
- **Buffer letter omission.** su + u → suyu (buffer y); ev + i → evi (no
  buffer). One rule: buffer only between vowels.
- **-de/-da spacing.** Locative suffix attaches (evde); the conjunction 'de'
  stands alone (Ali de geldi). Same sounds, different words.
- **Question particle harmony.** geliyor mu? / gördün mü? — mi harmonizes and
  stands apart.
- **Evidential -mış misuse.** Using -di for hearsay: "it apparently rained" =
  yağmur yağmış, not yağdı. English has no category here — teach with
  gossip/news framing.
- **SOV avoidance.** English order leaks in: keep the verb final in neutral
  sentences; word order shifts mark emphasis, not default.
