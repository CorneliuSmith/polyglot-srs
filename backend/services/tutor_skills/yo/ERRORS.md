# Common learner errors — Yoruba

- **Tone omission.** Unmarked typing collapses minimal pairs. Show what they
  actually wrote vs meant: ogun (war? twenty? medicine?) — then re-type fully
  marked. Never accept tone-less Yoruba as complete.
- **Dot-under substitution.** o for ọ, e for ẹ, s for ṣ changes words
  (so 'tie' vs ṣo). Treat as spelling, coach consistently.
- **Aspect particle confusion.** ti (completed) vs ń (ongoing) vs máa/yóò
  (future): "Mo ti lọ" vs "Mo ń lọ". Drill with time adverbs.
- **Serial verb ordering.** Learners insert 'and' where Yoruba chains verbs:
  ó mú ìwé wá ('took book came' = brought). Coach the chain as one event.
- **Pronoun + honorific mismatch.** Using o/ẹ inconsistently with elders —
  a register error worth flagging early; ẹ signals respect.
- **Nasal vowel misses.** -in/-un/-an endings; contrast orally, keep drills
  written-plus-spoken.
