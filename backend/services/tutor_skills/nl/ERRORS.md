# Common learner errors — Dutch

- **No inversion after a fronted element.** "Morgen ik ga…" — the verb owns
  position two: Morgen ga ik. Rebuild with the slots visible.
- **Verb not final in subclauses.** "…omdat ik ben ziek" → omdat ik ziek
  ben. Especially after omdat/dat/als.
- **want/omdat word order swapped.** want keeps main-clause order; omdat
  sends the verb to the end. Same meaning, different grammar.
- **niet for geen.** "Ik heb niet een auto" → Ik heb geen auto. If English
  says "no X / not a X", Dutch says geen.
- **The -e exception missed.** "een grote huis" → een groot huis (een +
  het-word + singular drops the -e). But: een grote man (de-word).
- **Participle as second verb in perfect+modal.** "Ik heb niet gekund
  slapen" → Ik heb niet kunnen slapen. Double infinitive, always.
- **Separable prefix left attached.** "Ik opsta om zeven uur" → Ik sta om
  zeven uur op. Main clause splits it; the prefix lands last.
- **-t kept in jij-inversion.** "Bent jij…?" → Ben jij…? Only jij/je
  triggers the drop, and only when it follows the verb.
- **hoeven used positively.** hoeven lives in negation only: Je hoeft niet
  te komen. Positive obligation is moeten.
- **er omitted.** "Ik heb drie" → Ik heb er drie. Quantities and unstressed
  places need er; leaving it out is the top intermediate tell.
- **False friends.** eventueel = possibly (not eventually), actueel =
  current (not actual), slim = smart (not slim). Flag them on sight.
